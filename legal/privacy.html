1. Information We Collect

  • Account information: name, email, password hash, session metadata.
  • API information: API key prefix/hash, request metadata, model, token counts, latency, errors, usage charges.
  • Billing information: order status, credits, payment provider references. We do not store full card numbers.
  • Technical information: IP address, user agent, logs, security events.

2. API Content

API prompts and responses may be processed by upstream model providers to fulfill requests. We may retain limited logs for debugging, security, abuse prevention, and billing unless otherwise configured.

3. How We Use Information

  • Provide and operate the API service.
  • Authenticate users and secure accounts.
  • Track usage, billing, credits, and costs.
  • Detect abuse, fraud, or policy violations.
  • Improve reliability and support users.

4. Sharing

We may share data with upstream model providers, payment providers, hosting providers, analytics/logging providers, and legal authorities when required.

5. Security

We use hashed API keys, session tokens, and access controls. No system is perfectly secure; users should avoid submitting unnecessary sensitive data.

6. Retention

We retain account, billing, and usage records as needed for service operation, security, legal, and accounting purposes.

7. Your Rights

You may request access, correction, deletion, or export of your account information where applicable.
