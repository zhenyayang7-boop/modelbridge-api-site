Privacy Policy

Draft MVP privacy policy for account, API, usage, and billing records.

Information collected

Account information, API key prefix/hash, request metadata, model, token counts, latency, errors, usage charges, billing records, IP address, user agent, logs, and security events.

API content

API prompts and responses may be processed by upstream model providers to fulfill requests. Limited logs may be retained for debugging, security, abuse prevention, and billing unless otherwise configured.

Use of information

We use information to operate the API service, authenticate users, secure accounts, track credits and costs, detect abuse or fraud, and support users.

Sharing

Data may be shared with upstream model providers, payment providers, hosting providers, analytics/logging providers, and legal authorities when required.

Security and retention

We use hashed API keys, session tokens, and access controls. Records are retained as needed for service operation, security, legal, and accounting purposes.

Rights and contact

You may request access, correction, deletion, or export where applicable. Contact: support@modelbridgeapi.com
