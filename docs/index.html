Python

OpenAI-compatible client

Replace the key with the token created in the console.

pip install openai

from openai import OpenAI

client = OpenAI(
    api_key="YOUR_MODELBRIDGE_API_KEY",
    base_url="https://api.modelbridgeapi.com/v1"
)

response = client.chat.completions.create(
    model="deepseek-chat",
    messages=[{"role": "user", "content": "Say hello"}]
)
print(response.choices[0].message.content)

REST

Chat completions

The hosted API follows the OpenAI-compatible /v1 shape for supported routes.

curl https://api.modelbridgeapi.com/v1/chat/completions \
  -H "Authorization: Bearer YOUR_MODELBRIDGE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "deepseek-chat",
    "messages": [{"role":"user","content":"Hello"}]
  }'

Route policy

Use compliant upstream APIs, not shared subscriptions.

ModelBridge is not designed around account pools, subscription resale, browser reverse engineering, or unlimited-access claims. Routes should be backed by official APIs, cloud providers, or verified partner channels.
